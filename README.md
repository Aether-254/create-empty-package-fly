# Create: Empty Package — Create Fly

Allows Create Fly Packagers to produce an empty package when the target is empty, missing, or not an inventory.

## Supported branches

`fabric/26.2, fabric/26.1.2, fabric/26.1, fabric/1.21.10, fabric/1.21.8, quilt/1.21.11, quilt/1.21.10, quilt/1.21.8`

Each loader/Minecraft combination is maintained on its own branch. Select the matching branch before building or contributing.

## Building

Run `./gradlew build` on Linux/macOS or `gradlew.bat build` on Windows. Artifacts are written to `build/libs`.
